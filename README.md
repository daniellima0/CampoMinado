# Minesweeper

## Introduction
This project was built using __Java__ and Eclipse IDE and consists on the classic Minesweeper game<br>

## Key Concepts
* Object Oriented Programming (OOP)
* GUI Widget Toolkit
    * Swing
    * Abstract Window Toolkit (AWT)
